# Конспект: Функции в Python (Лекция 5)

**Функция** — блок кода для выполнения задачи. Объявляется через `def`, после заголовка двоеточие, тело с отступом 4 пробела.

**Параметры vs аргументы:** параметры — в определении, аргументы — при вызове. Виды: позиционные, именованные, со значением по умолчанию. `*args` — произвольное число позиционных, `**kwargs` — именованных.

**return** завершает функцию и отдаёт результат. Без return возвращается None. Можно вернуть несколько значений кортежем.

**Область видимости:** локальные переменные не видны снаружи. Глобальные читаются свободно, для записи нужен `global`. Вложенные функции видят внешние переменные через замыкание.

**Рекурсия:** функция вызывает саму себя, обязателен базовый случай. Лимит глубины — 1000. Примеры: факториал, Фибоначчи, обход дерева.

**Lambda:** анонимные функции через `lambda`, одно выражение. Применяются с `map`, `filter`, `sorted`.
