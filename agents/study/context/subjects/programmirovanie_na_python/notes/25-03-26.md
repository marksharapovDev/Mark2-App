# Лекция 5: Функции в Python

## Что такое функция

Функция — это блок кода, который выполняет определённую задачу. Функции позволяют разбить программу на логические части и избежать дублирования кода. В Python функция объявляется с помощью ключевого слова def.

## Синтаксис

Функция состоит из заголовка (def, имя, параметры в скобках) и тела (отступ 4 пробела). После заголовка ставится двоеточие. Тело функции выполняется при каждом вызове.

## Параметры и аргументы

Параметры — это переменные в определении функции. Аргументы — конкретные значения при вызове. Бывают позиционные и именованные аргументы. Можно задавать значения по умолчанию. Через *args передаётся произвольное количество позиционных аргументов, через **kwargs — именованных.

## Возвращаемое значение

Оператор return завершает функцию и возвращает результат. Если return нет, функция возвращает None. Можно возвращать несколько значений через кортеж.

## Область видимости

Переменные внутри функции — локальные, не видны снаружи. Глобальные переменные доступны для чтения, но для изменения нужно ключевое слово global. Вложенные функции видят переменные внешней функции через замыкание.

## Рекурсия

Функция может вызывать саму себя. Обязательно нужен базовый случай для остановки. Классические примеры: факториал, числа Фибоначчи, обход дерева. Рекурсия ограничена глубиной стека (по умолчанию 1000 вызовов).

## Lambda-функции

Анонимные функции создаются через lambda. Используются для коротких выражений, часто с map, filter, sorted. Не могут содержать сложную логику — только одно выражение.
